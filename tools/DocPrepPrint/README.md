# Doc Prep & Print — Reference Files

These are the **original standalone scripts** that were refactored and integrated into
[Engineering Tool Hub](../../README.md). They are kept here as a reference only.

## Files

| File | Description |
|------|-------------|
| `DocPrepPrint.py` | Original standalone doc prep and print script |
| `DocPrepPrint_Test(makesPDFs).py` | Variant that saves PDFs locally instead of printing — used for testing without a printer |

## Notes

- All active functionality lives in [`app.py`](../../app.py) at the repo root.
- Do **not** run these scripts directly — use Engineering Tool Hub instead.
- These files are not included in the PyInstaller build.
